### TELEM1, TELEM2 ports

|Pin|Signal|Volt|
|---|---|---|
|1 (red)|VCC|+5V|
|2 (blk)|TX (OUT)|+3.3V|
|3 (blk)|RX (IN)|+3.3V|
|4 (blk)|CTS|+3.3V|
|5 (blk)|RTS|+3.3V|
|6 (blk)|GND|GND|
